# PASSWORD-MANAGER

Password-manager es un gestor de contraseñas que utiliza la encriptación por bloques.

# Instalación

Utiliza sudo para su configuracion correcta
## Linux
```bash
sudo ./setup
```

# Uso

## Linux
```bash
sudo pssmanager
```
